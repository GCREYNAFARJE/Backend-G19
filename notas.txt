pip list                     > lista todas las librerias de python
pip freeze                   > lista todas las librerias instaladas manualmente de python
pip install NOMBRE_LIBRERIA  > instala la libreria y si no la encuentra lanza un error 
python -m venv NOMBRE_ENTORNO > crea un entorno en la ubicacion especificada
source ENTORNO/scripts/activate > levanta el entorno para ser usado en terminales
source ENTORNO/bin/activate > levanta el entorno en S.O. Linux
ENTORNO/scripts/activate > levanta el entorno usando terminales POWERSHELL | CMD
Para salir del entorno "deactivate" > desactiva el entorno
pip freeze > requirements.txt > lista las librerias y lo guarda en el archivo requirements.txt
