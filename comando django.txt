django-admin startproject NOMBRE_PROYECTO > creara el proyecto dentro de la carpeta con ese nombre
django-admin startapp NOMBRE_APLICACION   > creara la aplicacion dentro del proyecto
python manage.py makemigrations NOMBRE_APP --name NOMBRE_MIGRACION > lee los modelos creados / cambiados y crea la migracion correspondiente
python manage.py migrate > ejecuta las migraciones en la bd y las que ya estan, las saltan
python manage.py showmigrations > mostrar todas las migraciones del proyecto y las que ya esten ejecutadas se pondra un [X] y las que no [ ]
